Project on Console


What is Visual Studio?
Visual Studio (Code) is a progrmme which helps you to write Javascript commands (easily).

How to set it up?
Step 1: Install (Download) Visual Studio.
Step 2: Open it.
Step 3: Go to Desktop.
Step 4: Create a new folder.
Step 5: Give it a name of your choice (but you should remember the name!).
Step 6: Double click on that folder, so as to open it.
Step 7: Create another new folder (inside that folder).
Step 8: Rename it.
Step 9: Doublle click on this (new created) folder.
sep 10: Come back to the Visual Studio.
Step 11: Click on the 'File' option, situated on the top left corner.
Step 12: Click on the 'New File' option.
Step 13: Again click on 'File' option.
Step 14: Click on 'Save As' option.
Step 15: Start writing the code in the file you created just now!

What is Console Screen?
Console Screen helps us in finding the errors, in the code we write, very easily. It also tells in which line is the error situated. You can also see the output sometimes, but only if write the command 'console.log'!

How does it help us?
As I also told earlier, console helps us to find the errors. And if you want to add a code directly in the console screen; then you don't have change the tabs {from visual studio to website (console)}.

How to open console screen?
Step 1: Right click on any blank space (on the website screen).
Step 2: Click on 'Inspect' option.
Ste 3: On the top of that screen, you'll see a toolbar.
Step 4: Click on the 'Console' option.
Step 5: Here opens the Console Screen.

How to print anything on console?
You will notice a cursor (i.e. used in text-inputs) blinking on the console. You will notice that cursor below your code (output). You don't have to click anywhere. Just start Typing!
